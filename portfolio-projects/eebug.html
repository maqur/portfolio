EEBug

Overview

For our first year group project we had to design and build a line following robot. The specification required the robot to follow an arbitrary line that had a specified maximum turning angle and could also have short breaks in it. At the end of the line it had to continue straight for further 20 cms and then spiral inwards for at least one complete rotation.

Implementation

There were two ways in which this could be acheived, analogue in which the whole design was based on components like transistors, resistors, etc and digital in which a microcontroller was used for all the required login. My group decided to use a microcontroller for the logic and state machine. The software was written and compiled using the arduino IDE. We used pulse width modulation to control the speed of motors. Infrared sensors were used for detecting the line and thresholds were set for the readings.

I was mainly involved in writing the state machinek software and designing the resistor circuit for connecting the sensors.
